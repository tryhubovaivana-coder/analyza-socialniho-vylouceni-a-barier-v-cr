# Analýza úspěšnosti crowdfundingových kampaní v sociálním sektoru

Vítejte u mého druhého projektu! Tento repozitář je zaměřen na makro analýzu komunitního financování (crowdfundingu) pro sociální podniky, charitativní projekty a neziskové organizace. Cílem je zjistit, jaké faktory rozhodují o tom, zda kampaň získá potřebnou finanční podporu od veřejnosti.

🎯 Vize projektu
Malé rozpočty sociálních podniků vyžadují přesné cílení. Analýzou historických dat z crowdfundingových platforem chci identifikovat klíčové vzorce úspěchu – od správného načasování startu kampaně, přes optimální nastavení výše cílové částky, až po dynamiku příspěvků v čase. Tento projekt pomůže sociálním organizacím lépe plánovat jejich fundraisingové strategie.

🛠️ Budoucí nástroje
* SQL pro segmentaci kampaní podle témat (ekologie, chráněné dílny, kultura) a výpočet průměrných darů.
* Python (Pandas/Seaborn) pro korelační analýzu mezi délkou kampaně, počtem odměn a výsledným úspěchem.
* Power BI pro tvorbu strategického přehledu, který ukáže ideální parametry pro spuštění nové kampaně.
